OGD Automation Badi Daten
==============

In diesem Verzeichnis ist der Code für zahlreiche Automationen, welche mit GitHub Actions gesteuert werden. Hier werden nur Daten bestimmter Datensätze aktualisiert. 

> **Wichtig:** Dieses Repo verwendet einige Hilfsskripte von https://github.com/opendatazurich/opendatazurich.github.io/tree/master/automation. Detaillierte Informationen zum allgemeinen Setup finden sich auch dort. 

Die Details zum Update der Badi-Datensätze sind in den Unterordnern von diesem Repo:
- [badi_aktuell_crowdmonitor](badi_aktuell_crowdmonitor/README.md)
- [badi_besuch_ase](badi_besuch_ase/README.md)
